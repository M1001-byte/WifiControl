# WifiControl

Connect to wifi using w/s and up/down, its very easy. :)



## Requirement

Required iwd. Use iwctl to get networks and connect.
```
sudo pacman -S iwd
```
## Installation
```bash
go install github.com/M1001-byte/WifiControl
```
    
## Usage
```bash
WifiControl wlan0 // network interfaz
```
